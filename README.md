# 📌 Project Overview

**ShopSphere E-Commerce Analysis** is a complete Business Intelligence project that analyzes an e-commerce company's sales performance using **Python, SQL, and Power BI**.

The project transforms raw transactional data into actionable business insights by analyzing:

- 📈 Revenue Trends
- 💰 Profit Analysis
- 📦 Product Performance
- 👥 Customer Segmentation
- 🛍 Brand Analysis
- 💳 Payment Behaviour
- 📊 Interactive Dashboard

---

# 🎯 Business Problem

The company wants to answer questions like:

- Which products generate the highest revenue?
- Which categories are growing?
- Which brands perform best?
- Which payment methods are most popular?
- How are customers behaving?
- Which customer segment contributes most revenue?
- What is the Average Order Value (AOV)?
- How did sales change between 2021 and 2022?

# 📌 Key Business Insights

✅ Total Revenue exceeded **4 Billion**

✅ Profit reached **1.34 Billion**

✅ Samsung generated the highest revenue.

✅ Apple recorded the highest Average Order Value.

✅ COD is the dominant payment method.

✅ Entertainment category showed the highest growth.

✅ Books and Others experienced declining sales.

✅ Revenue and profit peaked during late 2022.
